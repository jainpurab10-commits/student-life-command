About Student Life Command Center

I built this website as my first complete web development project using only HTML and CSS. The idea was simple: make one place where I can organise my college life properly.

Why I Made This

Most of the time, my timetable, assignments, fees, notes and exam dates were all scattered in chats, photos and rough pages. I wanted something cleaner.

So instead of making a basic portfolio or a random demo site, I decided to build a small “control center” for student life – something I can actually use every day.

The Problem

Timetable screenshots get lost, assignments are announced in class and forgotten, attendance is only checked at the end, and money just disappears without any record.

The Idea

Put everything in one website: timetable, assignment planner, attendance tracker, expense planner and a page for links and notes. Keep it simple, clean and printable.

Who It’s For

Any student can use this format. You just change the subject names, dates and links, and it works for any course or semester.

What the Site Includes

  • Weekly timetable and exam schedule pages.
  • Assignment and test planner with due dates.
  • Subject-wise and daily attendance pages.
  • Daily expense log and monthly budget planner.
  • A resources page with useful links and downloads.

Tech Used

This version is kept fully static on purpose to match my current syllabus.

  • HTML5: structure, tables, forms and semantic tags like <header>, <nav>, <main> and <section>.
  • CSS3: layout with flexbox and grid, spacing, fonts and simple responsive design.
  • No JavaScript / backend yet: data is meant to be written directly or used in printed form.

Future Plans

Once I move ahead in web development, I can turn this into a proper web app.

  • Automatically calculate attendance and totals using JavaScript.
  • Save all data in localStorage or a database.
  • Add login so each student gets their own dashboard.
  • Give options to export or print clean PDFs.
